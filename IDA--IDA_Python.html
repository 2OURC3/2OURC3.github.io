IDA Python


Make your reversing life easier with IDA Python.

Since some years, you can use Python with IDA to run different python script inside IDA, which is a great thing ! You can find the official API documentation here :
https://hex-rays.com/products/ida/support/idapython_docs/

How to :
- There are different ways to run your Python code into IDA. First, you can use the built-in interpreter in the bottom panel.
- You can either load the python script directly into the menu File > Script file.
- Or you can write a Python script inside IDA by clicking File > Script Command.
