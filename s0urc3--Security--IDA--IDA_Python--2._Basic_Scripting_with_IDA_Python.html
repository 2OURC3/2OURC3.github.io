Automating IDA Trough IDAPython :

The switch -OIDAPython:/path/to/python/script.py can be used to tell IDAPython which script to run on start up. Another switch is -A which will instruct IDA to run
in batch mode. With those two options combined it is possible to auto-analyze a binary and run a Python script.
